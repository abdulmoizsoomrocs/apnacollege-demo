# apnacollege-demo

A minimal static website built as a first Git repository project. It includes:

- **Home page (`index.html`)** with a welcome message and call-to-action button.
- **Contact page (`contact.html`)** featuring a simple form (name, email, message) and navigation link back to the home page.

## How to view locally
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/apnacollege-demo.git
   ```
2. Open `index.html` in your browser to view the site. You can navigate to the contact page via the on-page button or by opening `contact.html` directly.
